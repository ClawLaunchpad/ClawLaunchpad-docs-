## 🚀 Quickstart (1 minute)

1. Go to [clawlaunchpad.com](https://clawlaunchpad.com)
2. Choose a workflow (code ship, growth campaigns, ops reporting)
3. Click **Deploy** – we handle setup, security, and ClawRouter integration.

## 💼 Workflows

- **Ship Code** – git push → agent review/deploy.  
- **Growth Campaigns** – monitor signals → auto‑post/engage.  
- **Ops Reporting** – pull data → generate reports → share.  

## 🔒 Security First

- Sandboxed execution  
- ClawRouter for cost‑optimized model routing  
- USDC x402 payments (coming soon)
